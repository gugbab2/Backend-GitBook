# 객체지향의 4대 특성

## 객체와 클래스를 먼저 이해하자&#x20;

* 개인적인 특징을 가질 수 있다면 객체\
  \-> 경호는 안경을 쓴다.
* 개인적은 특징을 가질 수 없다면 클래스\
  \-> 사람은 안경을 쓴다?

## 추상화 : 모델링

* 객체지향의 추상화는 곧 모델링이다. \
  \-> 구체적인 것을 분해해서 관심 영역(애플리케이션 영역) 에 있는 특성만 가지고 재조합하는 것 \
  \-> 병원 / 은행 애플리케이션에서 사람을 정의하는 특성이 다르다. \
  \-> 클래스 설계를 위해서 객체의 정의부터 정해야 한다.&#x20;
