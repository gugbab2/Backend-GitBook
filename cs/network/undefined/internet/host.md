* 컴퓨터가 네트워크에 연결되어 있다고 가정하면, **컴퓨터를 이제는 호스트라고 부른다!**\
  &#xNAN;**-> 호스트 : 인터넷에 연결된 컴퓨터**
* 호스트는 둘로 나뉜다.
  * 스위치 : 네트워크 그 자체를 이루는 호스트(컴퓨터)\
    -> **Infra Structure** : Router, IPS, Tab ...
  * 엔드포인트 : 스위치를 제외한 호스트(컴퓨터)\
    -> Clienct, Server, Peer ...
